# TestVibeGame - Setup & Instructions

I've set up the core mechanics for your 3rd person spear-throwing game!

## Included Files
- **Player.gd / Player.tscn**: The player character with movement and over-the-shoulder camera.
- **Spear.gd / Spear.tscn**: The spear projectile physics.
- **World.tscn**: A simple test level with a floor and lighting.

## How to Play
1. Open the project in Godot.
2. Open **World.tscn**.
3. Press **F6** (or click the "Run Current Scene" button) to play the scene.

## Controls
- **WASD / Arrow Keys**: Move the character.
- **Mouse**: Rotate the camera.
- **Left Click**: Throw a spear.
- **Space**: Jump.
- **Esc**: Release the mouse cursor.
